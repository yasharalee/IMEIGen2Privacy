Privacy Policy for IMEI Generator

Effective date: 07/10/2024

Information Collection and Use

We do not collect, store, or share any personal data from users of the Service. The Service operates entirely within the user's browser and does not transmit any data to external servers or third parties.

Permissions

The Service requires the following permissions to function:

  • activeTab: Allows the extension to access the current tab to execute the IMEI generation and insertion process.
  • scripting: Enables the extension to execute scripts in the context of web pages to insert the generated IMEI numbers directly into input fields.
  • contextMenus: Used to create custom context menu options for selecting device models.
  • clipboardWrite: Not used in the current version. (Only include this if it is indeed used)
  • Host Permissions (http://*/* and https://*/*): Allows the extension to operate on all web pages to provide the IMEI generation functionality.
